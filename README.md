**Author:** Str3llbat  
**Version:** 1.0

---

## 📌 About

A simple GUI script for Roblox that adds:

- Infinite Yield (command menu)
- Fly mode (toggle with F)
- Fling players by username
- Speed / Jump modifiers
- NoClip

⚙️ Controls
Button	Action
Inf Yield	Opens IY (type ; in chat)
Fly	Toggle flight (press F)
Fling	Launch a player (enter name)
Speed x2	Double walkspeed
Jump x2	Double jump power
NoClip	Walk through walls

Reset to disable fly or smthing
